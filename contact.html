联系我们

欢迎反馈攻略错误、图片问题、失效链接或你希望看到的新内容。

推荐方式：在项目的 GitHub Issues 中提交问题，便于记录、跟踪和回复。

前往 GitHub Issues

反馈时请包含

  • 发生问题的页面地址。
  • 问题截图或具体棋盘局面。
  • 你认为正确的走法或需要补充的信息。
  • 设备和浏览器信息（仅在显示问题时需要）。

隐私提醒

请勿在公开反馈中提交身份证、银行卡、住址、电话号码、AdSense 付款资料或账号验证码。
