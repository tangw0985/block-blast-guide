隐私政策

本政策说明 Block Blast 攻略站如何处理访问数据、Cookie 和第三方广告服务。

更新日期：2026年6月22日

我们收集的信息

本网站本身不要求用户注册账号，也不会主动收集身份证、银行卡或精确位置等敏感个人信息。服务器、托管平台和统计服务可能自动记录浏览器类型、设备类型、访问时间、来源页面、IP 地址及页面访问情况。

Cookie 与 Google AdSense

本网站使用 Google AdSense 展示广告。Google 及其合作伙伴可能使用 Cookie，根据用户访问本网站或其他网站的情况投放、衡量和个性化广告。对于欧洲经济区、英国和瑞士访客，网站使用 Google 认可的用户意见征求机制展示相关选择。

用户可以在浏览器中管理或删除 Cookie，也可以访问 Google 的广告设置页面管理个性化广告偏好。

第三方服务

  • Google AdSense：广告投放与效果衡量。
  • GitHub Pages：网站托管与静态内容分发。
  • Giscus（启用时）：基于 GitHub 的评论功能。

外部链接

网站可能链接到应用商店、GitHub 或其他第三方网站。第三方网站拥有各自的隐私政策，本网站不控制其数据处理方式。

儿童隐私

本网站不以主动收集儿童个人信息为目的。如果发现不当收集情况，请通过联系页面通知我们。

联系我们

对本政策有疑问，可通过联系页面提交反馈。
